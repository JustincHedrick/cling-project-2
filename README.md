<h2>Getting Started </h2>

<h3>Instructions </h3>
<details open>
<summary>Accessing Cling</summary>
<ol>
<li>A user is prompted at the landing page to sign in using Google OAuth.</li>
<li>After authorizing, the user is redirected to the Cling home page.</li>
</ol>
</details>

<details>
<summary>Using Cling</summary>
<ol>
<li>The home page gives the user a chance to see other users created plans sitewide.</li>
<li>By clicking the 'details' link, a user can review and add comments to any side-wide users plan</li>
</ol>
</details>

<details>
<summary>Create with Cling</summary> 
<li>On the sites NAV bar, the user can click on the 'create' link</li>
<li>The user will be redirected to a form</li>
<li>After the user successfully fills out the create form, clicking add plan will then redirect the user to the newly create plan details</li>
</details>
